// 01 — Project Brief

The Illusion
of Alignment

As corporations rapidly deploy Local Language Models to manage sensitive internal workflows, the reliance on System Prompts for Role-Based Access Control has introduced structural vulnerabilities invisible to most security teams.

Our Senior Project, FuzzyAI, exposes these weaknesses systematically. We built an automated pipeline focused on Prompt Extraction — using static, format-forcing algorithms to prove that enterprise AI defenses can be consistently dismantled.

The attack surface is deceptively simple. When forced to prioritize schema compliance over safety alignment, the model yields — leaking restricted database hashes, salary data, and master access codes.

// 02 — Extraction Pipeline

How The Attack Works

STEP_01 / FORMAT_FORCING

Format Forcing

Inject payloads demanding raw API JSON responses. Malicious requests are wrapped in hypothetical framing, exploiting the model's instruction-following priority over safety alignment rules.

STEP_02 / GUARDRAIL_COLLAPSE

Guardrail Collapse

The model's semantic filters fail under schema pressure. Structural format requirements override embedded safety rules — RBAC restrictions are bypassed silently, without triggering refusals.

STEP_03 / DATA_EXFILTRATION

Data Exfiltration

The AI outputs restricted arrays, master bypass keys, salary data, and hidden system instructions directly to the attacker. The breach is logged in real-time via async I/O.
