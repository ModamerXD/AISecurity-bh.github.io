01. Project Brief

The Illusion of Alignment

As corporations rapidly deploy Local Language Models (LLMs) to manage sensitive internal workflows, the reliance on System Prompts for Role-Based Access Control (RBAC) has introduced critical vulnerabilities.

Our Senior Project, FuzzyAI, exposes these structural weaknesses. We built an automated pipeline focused specifically on Prompt Extraction. By executing static, format-forcing algorithms, the framework proves that enterprise AI defenses can be consistently dismantled to leak restricted database hashes and access codes.

Framework Specs

  • Target Simulation: Local Ollama environments mimicking restricted HR Vaults.
  • Automated Judge: Custom anti-hallucination cloud prompting to eliminate false positives.
  • Fault Tolerance: Progressive async I/O writing for immediate data retention.
02. Methodology

The Extraction Pipeline

1. Format Forcing

Inject payload demanding raw API JSON responses, wrapping malicious requests in hypothetical scenarios.

2. Guardrail Collapse

The model's semantic filters fail, prioritizing the required formatting schema over its safety alignment rules.

3. Data Exfiltration

The AI outputs restricted arrays, master bypass keys, and hidden instructions directly to the attacker.

03. Project Findings

The Critical Threshold

35

%
