**FuzzyAI** is a hybrid-cloud cybersecurity project engineered to evaluate LLM constraints, exploit format-forcing vulnerabilities, and bypass system-prompt guardrails (RBAC).

## 📌 Project Overview
As corporations rapidly deploy Local Language Models (LLMs) to manage sensitive internal workflows, the reliance on System Prompts for Role-Based Access Control (RBAC) has introduced critical vulnerabilities. 

Our framework automates the process of "Prompt Extraction." By executing static, format-forcing algorithms, we proved that out-of-the-box enterprise AI defenses can be consistently dismantled to leak restricted database hashes and access codes.

### Key Features
* **Target Simulation:** Local Ollama environments mimicking restricted HR Vaults.
* **Automated Judge:** Custom anti-hallucination cloud prompting to eliminate false positives.
* **Fault Tolerance:** Progressive async I/O writing for immediate data retention.
